# ♟ Chess Game

A fully interactive browser-based Chess Game built using HTML5, CSS3, and JavaScript.
This project simulates the classic chessboard with complete piece movement, capturing mechanics, turn-based gameplay, and special moves like castling.

## Features

- Interactive 8×8 chessboard UI
- Complete chess piece setup
- Valid move highlighting
- Piece movement and capturing system
- Turn-based gameplay (White vs Black)
- Castling support
- Light and Dark theme support
- Smooth animations and highlighted move effects
- Responsive design for better user experience

---

## Technologies Used

- HTML5 – Structure of the chessboard
- CSS3 – Styling, animations, and themes
- JavaScript – Game logic and functionality
- jQuery – DOM manipulation and event handling

---

## 📂 Project Structure

```bash
Chess-Game/

│── style.css        # Main styling file
│── script.js        # Chess game logic
│── dark.css         # Dark mode theme
│── light.css        # Light mode theme
│── README.md        # Project documentation
````

---

## 🎮 How to Play

1. Select a chess piece.
2. Valid moves will be highlighted.
3. Click on a highlighted cell to move.
4. Capture opponent pieces by clicking on them.
5. Players alternate turns.
6. Use strategy to win.
